1. The login page

2. The dashboard

So I just logged in with the placeholder values.

Welcome page. Nothing interactive except product and an “Admin Console” button in the bottom right — obviously our target.
Clicking it → 403 Forbidden.

Noticed a signed session cookie. Modifying it is impossible without the secret, so I just deleted the cookie entirely → And Admin Console is accessible!

4. Set up ZAP

So, We need to fuzz an Endpoint, We can do that through ZAP easily.

We select "proudcts" → Add → Strings → Paste our payload.

Any page other than the product pagewe get redirected to the login. Could get annoying pretty quickly, So lets make a filter for it.

We go to Message Processors → Add → Type: Tag Creator → Regex: Found → Tag: Bad

And we find our endpoint!
